# Contribution Guidelines for Developers

Welcome to the Agentic AI project! We appreciate your interest in contributing to our project. Below are the contribution guidelines that all contributors should follow.

## Development Setup
1. **Clone the Repository**:  
   Run the following command to clone the repository:
   ```bash
   git clone https://github.com/kasper-openUsource/agentic-ai.git
   cd agentic-ai
   ```

2. **Set Up the Environment**:  
   Ensure you have [Node.js](https://nodejs.org/) installed. Then, install the required dependencies:
   ```bash
   npm install
   ```

3. **Run the Application**:  
   To run the application in development mode, use:
   ```bash
   npm start
   ```

## Testing Requirements
- All contributions must be accompanied by tests. 
- To run the tests, use:
  ```bash
  npm test
  ```
- Ensure that all tests pass before submitting your contribution.

## Code Style Guidelines
- Follow the [Airbnb JavaScript Style Guide](https://github.com/airbnb/javascript).
- Use ESLint to check your code:
  ```bash
  npx eslint .
  ```
- Format your code using Prettier before submitting:
  ```bash
  npx prettier --write .
  ```

Thank you for contributing, and we look forward to your input!
